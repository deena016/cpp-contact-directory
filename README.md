# cpp-contact-directory
C++ contact directory application that stores, searches, removes, and exports contacts using a map-based data structure.

This project implements a simple contact directory application in C++. The directory stores contacts using a map<string, Contact>, allowing efficient searching, insertion, and deletion of contacts by name. The program also supports exporting contacts to CSV files and performing partial name searches.

Features:
- Add new contacts
- Prevent duplicate contacts
- Remove contacts
- Search for a contact by name
- Perform partial name searches
- Export contacts to CSV files
- Automatically maintains alphabetical ordering using std::map

Concepts Demonstrated:
- C++ classes and structs
- STL containers (map, vector)
- File I/O (fstream)
- Operator overloading
- Basic data structure design
